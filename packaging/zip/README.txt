Hinton — 로컬·오프라인 AI 학습 도우미 (Gemma 4 E4B 모델 포함)
Hinton — local, offline AI study assistant (Gemma 4 E4B model included)

────────────────────────────────────────────────────────
실행 방법 / How to run
────────────────────────────────────────────────────────
1) 이 ZIP을 원하는 폴더에 압축 해제하세요.
   Extract this ZIP to any folder.

2) "Hinton" 폴더 안의  Hinton.exe  를 실행하세요.
   Open the "Hinton" folder and run  Hinton.exe

   → 첫 실행부터 실제 Gemma 4 E4B 모델이 로컬에서 동작합니다.
     인터넷·추가 설치·환경설정이 필요 없습니다.
   → Real Gemma 4 E4B runs locally from the first launch.
     No internet, no extra install, no configuration.

(선택) "바로가기 만들기.cmd" 를 더블클릭하면
       바탕화면 + 시작메뉴 바로가기가 생성됩니다.
(Optional) Double-click "바로가기 만들기.cmd" to create
           Desktop + Start Menu shortcuts.

────────────────────────────────────────────────────────
참고 / Notes
────────────────────────────────────────────────────────
• 대화·노트북·설정 데이터는  %LOCALAPPDATA%\Hinton  에 저장됩니다.
  Your chats/notebooks/settings live in  %LOCALAPPDATA%\Hinton
• 더 큰 12B(에스컬레이션) 모델이 필요하면
  "Hinton-12B-Setup" 플러그인 설치 프로그램을 실행하세요.
  For the larger 12B escalation model, run the
  "Hinton-12B-Setup" plugin installer.
• 제거: "Hinton" 폴더와 바로가기를 삭제하면 됩니다.
  Uninstall: delete the "Hinton" folder and the shortcuts.
